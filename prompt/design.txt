# UI 개선 및 리브랜딩 요청

## 목표
제공된 이미지(image_0.png)의 웹 애플리케이션 화면 디자인을 PwC(PricewaterhouseCoopers)의 공식 브랜드 스타일 가이드를 적용하여 완전히 새롭고 전문적으로 재구성하십시오. 원본의 모든 기능과 텍스트 콘텐츠를 유지하되, 레이아웃을 더 깔끔하고 조잡하지 않게 정리해야 합니다.

## 브랜드 가이드라인 (PwC 스타일)
1.  **색상 팔레트:**
    * **기본 텍스트 및 프레임:** 완전한 검은색(#000000) 또는 매우 짙은 회색(#2D2D2D)을 사용하십시오.
    * **포인트 컬러 (로고 그라데이션):** 아래의 핵심 색상을 그라데이션이나 포인트로 배합하여 사용하십시오.
        * Yellow: `#FFB600`
        * Tangerine: `#EB8C00`
        * Orange: `#D04A02`
        * Rose: `#DB536A`
        * Red: `#E0301E`
    * **강조 및 알림:** 중요한 상태 변경이나 경고에는 Red(#E0301E) 또는 Orange(#D04A02)를 사용하십시오.
    * **배경:** 깨끗하고 깔끔한 흰색(#FFFFFF) 또는 매우 연한 회색(#F8F9FA) 배경을 사용하십시오.
2.  **타이포그래피:**
    * 고급스럽고 전문적인 느낌의 산세리프 글꼴(예: 'PwC Sans' 또는 유사한 오픈소스 글꼴 'Inter', 'Noto Sans KR')을 사용하십시오.
    * 제목, 부제목, 본문 텍스트의 크기 계층을 명확하게 두어 가독성을 높이십시오.
3.  **레이아웃 및 간격:**
    * 정보의 계층 구조를 명확히 하십시오.
    * 충분한 여백(white space)을 사용하여 시각적 혼란을 줄이고 각 요소가 숨을 쉴 수 있게 하십시오.
    * 카드 기반 레이아웃을 활용하여 정보를 구획화하는 것을 권장합니다.
4.  **구성 요소 스타일:**
    * 모든 요소(카드, 버튼, 입력 필드 등)에 직각 또는 매우 미세한 둥근 모서리(border-radius: 4px 이하)를 사용하여 전문적이고 안정적인 느낌을 주십시오.
    * 미니멀한 디자인을 추구하며, 불필요한 그림자나 장식은 배제하십시오.

## 상세 화면 구성 요소 재정의

1.  **헤더 (최상단):**
    * 가장 좌측에 PwC 공식 로고를 배치하십시오.
    * 로고 옆에 제목 "PE 투자보고서 자동 생성 시스템"을 배치하십시오.
    * 그 아래 부제목 "투자위원회 수준의 보고서를 자동으로 생성합니다"를 더 작은 글씨로 배치하십시오.
2.  **단계별 탐색 바 (Step Indicator):**
    * 원본의 4단계 탭을 더 세련되게 재구성하십시오.
    * 완료된 단계(1, 2, 3)는 연한 회색 배경과 검은색 텍스트로 표시하십시오.
    * 현재 단계(4. 보고서 생성)는 PwC 로고 그라데이션(Yellow -> Orange -> Red)을 탭 전체 배경으로 적용하거나, 하단에 굵은 그라데이션 선을 배치하여 시각적으로 강조하십시오.
3.  **콘텐츠 영역 (4단계: 보고서 생성):**
    * 전체 영역을 하나의 깔끔한 카드 스타일 용기로 묶으십시오.
    * **'입력 요약' 섹션:**
        * 원본의 아코디언 스타일을 유지하되, 제목 줄을 PwC Black(#2D2D2D) 배경과 흰색 텍스트로 만들어 눈에 띄게 하거나, 카드 상단에 명확한 헤더로 배치하십시오.
        * 펼쳐졌을 때 내부 콘텐츠는 미니멀한 리스트 형식으로 정리하십시오.
    * **'에이전트 진행 상황' 섹션:**
        * 제목을 PwC Black(#2D2D2D) 또는 Orange(#D04A02) 색상으로 더 크고 굵게 표시하십시오.
        * 원본의 리스트를 더 깔끔하게 정리하십시오.
        * `Consulting Agent (전략): 완료`는 녹색(#28A745) 완료 아이콘과 함께 표시하십시오.
        * `Exit Strategy Agent (엑싯): 실행 중...`은 노란색(#FFB600) 또는 주황색(#EB8C00) 진행 아이콘과 함께 표시하십시오.
    * **상태 바 및 메시지:**
        * 메시지 "보고서를 생성하고 있습니다..."를 상태 바 바로 위에 배치하십시오.
        * 상태 바는 PwC 그라데이션(Yellow -> Orange -> Red)으로 채워지도록 디자인하십시오.
    * **'보고서 모드 선택' 섹션:**
        * 제목 "보고서 모드 선택"을 명확한 헤더로 배치하십시오.
        * 라디오 버튼과 텍스트를 더 세련된 디자인으로 재구성하십시오.
        * 각 옵션을 시각적으로 구획화된 별도의 미니 카드로 만들어 선택 시 테두리 그라데이션이 활성화되도록 할 수 있습니다.
        * 선택된 옵션(`IC Memo`)의 세부 설명 텍스트를 더 명확하게 표시하십시오.
4.  **버튼 영역 (최하단):**
    * 전체 카드의 우측 하단 또는 콘텐츠 영역 아래 중앙에 배치하십시오.
    * **`<- 이전` 버튼:** 깔끔한 테두리 버튼(outlined button) 스타일로, 검은색 테두리와 텍스트를 사용하십시오.
    * **`보고서 생성 ->` 버튼:** PwC 로고 그라데이션(Yellow -> Orange -> Red)을 버튼 전체 배경으로 적용하고 흰색 텍스트를 사용하여 가장 강력한 행동 유도 버튼(Primary CTA button)으로 만드십시오. 버튼을 클릭 가능한 주황색 계열로 유지하되, PwC 컬러를 적용하십시오.

## 기술 요구 사항
* 반응형 웹 디자인(Responsive Web Design)으로 구현하여 다양한 화면 크기에서도 깔끔하게 보이도록 하십시오.
* 클린 코드(clean code) 원칙을 따르고, 직관적인 CSS 클래스 이름을 사용하십시오 (예: `pwc-header`, `pwc-step-indicator`, `pwc-progress-bar`).
* 제공된 원본 이미지의 모든 텍스트 콘텐츠와 기능적 계층 구조를 정확하게 보존하십시오.
